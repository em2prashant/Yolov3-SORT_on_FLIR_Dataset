# Yolov3-SORT_on_FLIR_Dataset

## Description:
In this repository, a <a href = "https://pjreddie.com/media/files/papers/YOLOv3.pdf">"YOLOv3: An Incremental Improvement"</a> detector has been trained on the open sourced FLIR Dataset whose link is provided in the "Dataset" section below. Apart from training a detector, the task was also to track the detected objects which was carried out using the "Simple Online and Realtime Tracking (SORT)" algorithm. 

## Dataset:
Link: https://www.flir.in/oem/adas/adas-dataset-form/
